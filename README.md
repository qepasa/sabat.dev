## Installation
Standard python
```
git clone https://github.com/Cloud11665/sabat.dev
cd ./sabat.dev
python -m pip install -r ./requirements.txt
```
Pipenv
```
git clone https://github.com/Cloud11665/sabat.dev
cd ./sabat.dev
pipenv install
```
